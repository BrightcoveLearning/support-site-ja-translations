プラットフォームAPIコードサンプル

データ収集サンプル

このサンプルでは、​​データコレクタに情報を送信する方法を学習します。これにより、データコレクタでは表示されないビデオの分析データを取得できます Brightcove WebまたはSDKプレーヤー

リンクをダウンロード

このサンプルでは、 CMS API を使って高ビットレートの MP4 レンディションをダウンロードできるリンクを取得する方法について説明します。

iTunesのRSS Podcastフィード

このサンプルでは、 CMS API Apple iTunesのRSSポッドキャストフィードを生成します。

プレイリストからのMRSSフィード

このサンプルでは、 CMS API MRSS仕様に基づいてMRSSフィードを生成します。

MRSSジェネレータ

このサンプルでは、 CMS API MRSS仕様に基づいてMRSSフィードを生成します。

ビデオスキーマビルダ

このサンプルは、MicrodataまたはJSON-LD形式のVideoObjectスキーマを生成します。 あなたのビデオにスキーマを含めることで、検索エンジンインデクサーに情報が提供され、SEOが強化されます。

カスタムコンテキスト対応のエンコーディングプロファイルの作成

このサンプルでは、​ Ingest Profiles API を用いて​カスタム CAE プロファイルを作成する方法を学習します。

クライアント資格情報を作成する

このサンプルを使用すると、1つまたは複数のアカウントのクライアントクレデンシャルを作成し、選択したAPI権限をそれらに割り当てることができます。

アクセストークンを取得する

このサンプルは、アクセストークンを Brightcove OAuth API Webアプリケーションから、プロキシを介してリクエストを送信する
