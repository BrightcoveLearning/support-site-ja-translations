generate_dynamic_profile false CAEでは、このような出力の数とパラメータが、入力ファイルごとに動的に変更されることがあります。

outputs なし 出力設定の配列またはハッシュ。
region us ファイルが処理される地域：米国、ヨーロッパ、アジア、またはオーストラリア。
test false ジョブのテストモード（「統合モード」）を有効にします。
private false ジョブのプライバシーモードを有効にする。
download_connections 5 ダウンロード加速のために複数の同時接続を利用することがあります（状況によっては）。
pass_through なし このジョブの横に保存するオプションの情報。

grouping なし このジョブのグループ化されたレポート。
aspera_transfer_policy フェア Asperaファイル転送に使用可能な帯域幅を割り当てる方法。
transfer_minimum_rate 1000 データ転送の最低速度の目標速度（Kbps）。

expected_md5_checksum なし 入力ファイルの予想されるチェックサム。

一般的な出力設定

設定 デフォルト 説明
type 標準 出力するファイルのタイプ。
label なし この出力のオプションのラベル。
dynamic_profile_rendition なし dynamic_profile_renditionの設定では、このOutputが動的に生成されるCAEプロファイルの設定を使用するように指定します。具体的には、設定を使用するプロファイルのレンディションです。
url なし S3、クラウドファイル、GCS、FTP、FTPS、SFTP、Aspera、Azure、HTTP、またはRTMP URLを使用して、トランスコードされたファイルを配置します。
secondary_url なし S3、クラウドファイル、GCS、FTP、FTPS、SFTP、HTTP、RTMP、Aspera、またはAzureのURLで、トランスコードされたファイルを配置します。
base_url なし S3、クラウドファイル、GCS、FTP、FTPS、SFTP、Azure、またはAsperaのディレクトリURL。トランスコードされたファイルをファイル名なしで配置します。
filename なし 完成したファイルのファイル名。
package_filename なし パッケージ化された出力のファイル名。
package_format なし 出力ファイルに使用するZip / Packaging形式。
device_profile なし モバイルデバイスとの互換性に使用するデバイスプロファイル。
strict false 厳密モードを有効にします。
skip_video false ビデオトラックを出力しない。
skip_audio false オーディオトラックを出力しないでください。
source なし 別のジョブのラベルを参照し、入力ファイルの代わりにその出力によって作成されたビデオを処理に使用します。
credentials なし ニックネームで保存された資格情報を参照します。
generate_md5_checksum false 出力ファイルのMD5チェックサムを生成します。
parallel_upload_limit S30の3、他のすべての宛先の10。 試行する同時アップロードの最大数。
headers なし ファイルをアップロードするときにファイルとともに送信するHTTPヘッダー。

CAEの設定

設定 デフォルト 説明
min_renditions 1 生成するレンディションの最小数。
The minium number of renditions to generate 8 生成するレンディションの最大数。
min_resolution 192 x 108 ラダーの最小分解能。
max_resolution ソース解像度 ラダーに考慮する最大分解能。
max_frame_rate 30 使用可能な最大フレームレート。
keyframe_rate 0.5 キーフレームレート。 この値は、0.5または1のいずれかです。 SSAI対応コンテンツの場合、この値は1である必要があります。
fixed_frame_rate true すべてのレンディションで同じフレームレートを使用するようジェネレータに指示します。 もし false、ＸＮＵＭＸｐ未満の解像度でのＨ．ＸＮＵＭＸベースラインプロファイル表現は、半分のフレームレートが割り当てられる。
min_bitrate 200 使用可能な最小ビットレート[kbps]。
max_bitrate 15000 使用可能な最大ビットレート[kbps]。
max_first_rendition_bitrate 400 最初の演奏に使用できる最大ビットレート[kbps]。
min_granularity 30 レンディション間のビットレートの最小相対変化。
max_granularity 90 レンディション間のビットレートの最大相対変化。
video_codec h264 使用するビデオコーデック。
bitrate_cap_to_bitrate_ratio 1.3 VBR符号化に使用される平均ビットレートに対する最大ビットレートの比率。
buffer_size_to_bitrate_ratio 2.0 VBR符号化に使用される平均ビットレートに対するデコーダバッファサイズの比率。
bitrate_cap_offset 64 最大ビットレートを計算するときに定数が追加されます。
buffer_size_offset 85 VBVバッファサイズを計算するときに加算される定数です。
quality_rate_tune 0 品質対速度調整パラメータ。
select_baseline_profile_configuration true プロファイルで使用される少なくとも1つのレンディションがベースラインプロファイルになります。
video_configurations なし CAEエンコーディングの一部として優先的にまたは強制的に生成されるべき解像度、フレームレート、および関連するコーデック設定を記述する構造体の配列。

フォーマットとコーデック

設定 デフォルト 説明
format 出力ファイル名とビデオまたはオーディオコーデックによって決定されます。 それ以外の場合：mp4（標準出力の場合）; ts（セグメント化された出力用）。 使用する出力形式。
video_codec フォーマット、プロファイル、または audio_codec 。 存在しない場合はh264。 使用するビデオコーデック。
audio_codec フォーマット、プロファイル、または video_codec 。 何も提供されていなければaac。 使用するオーディオコーデック。

分解能

設定 デフォルト 説明
size なし 出力ビデオの解像度（WxH、ピクセル単位）。
width なし 出力ビデオの最大幅（ピクセル単位）。
height なし 出力ビデオの最大高さ（ピクセル単位）。
upscale false 入力が目標出力解像度よりも小さい場合は、出力をアップスケールします。
aspect_mode 保存する 入力ファイルのアスペクト比が目標の幅/高さアスペクト比と一致しない場合の対処方法
sample_aspect_ratio なし 出力ビデオに使用されるサンプル（ピクセル）アスペクト比。 サイズ、幅、高さ、および aspect_mode モードオプションは、すべて、ディスプレイの寸法ではなく、エンコードされたピクセル寸法を参照します。
force_interlace_headers false 一部のデコーダはインターレースされたコンテンツを要求しますが、プログレッシブコンテンツは正常に再生します。 この設定では、プログレッシブコンテンツのメタデータに「インターレース」ヘッダーが追加され、実際にインターレースされているように見えます。

レート制御

設定 デフォルト 説明
quality 3 最適なビデオビットレートを自動的に選択して、目標のビジュアル品質に合わせます。
video_bitrate なし 目標ビデオビットレート（kbps）。 特定のビットレートをターゲットにしない限り、品質設定を選択する必要はありません。
audio_quality 3 最適なオーディオビットレートを自動的に選択して、目的の音質に合わせます。
audio_bitrate なし 目標オーディオビットレート（kbps）。 あなたが audio_quality あなたが特定のビットレートを目標としない限り、設定します。
max_video_bitrate なし 最大平均ビットレート。
speed 3 ターゲットのトランスコード速度。 通常、エンコードが遅いほど、より高度な圧縮が可能になります。
decoder_bitrate_cap なし 最大ビットレートがデコーダバッファに供給される。 通常、ストリーミングや特定のデバイスをターゲットとするためのビデオに使用されます（例： Blu-Ray).
decoder_buffer_size なし デコーダバッファのサイズ。 bitrate_cap .
one_pass false ワンパスエンコーディングを強制します。
audio_constant_bitrate false 可能であれば、音声の固定ビットレートモードを有効にします。

フレームレート

設定 デフォルト 説明
frame_rate 元の（入力）フレームレート。 使用するフレームレート。
max_frame_rate なし 使用する最大フレームレート。
decimate なし 除数で入力フレームレートを下げます。
keyframe_interval 250 各キーフレーム間の最大フレーム数。 H.264ビデオの場合、キーフレームはIDRフレームです。
keyframe_rate keyframe_interval 250の 1秒あたりのキーフレームの数。
fixed_keyframe_interval false 固定キーフレーム間隔モードを有効にする（VP6、VP8、VP9、およびH.264のみ）。
forced_keyframe_interval なし 指定した間隔で強制的にキーフレームを適用します（H.264のみ）。
forced_keyframe_rate なし フレームレートを考慮して、1秒あたりのキーフレーム数を指定します（H.264のみ）。
generate_frame_index false ビデオフレームインデックスファイルを生成します。
frame_index_filename なし フレームインデックスのデフォルトのファイル名を上書きする generate_frame_index 有効になっています。

設定 デフォルト 説明

color_metadata 保存する 出力ビデオの入力からカラーメタデータ情報を保持または破棄します。
