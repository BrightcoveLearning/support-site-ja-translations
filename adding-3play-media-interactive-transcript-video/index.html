このトピックでは、3Playプラグインを使ってインタラクティブなトランスクリプトをに追加する方法を学びます。 Brightcove Player.

インタラクティブなトランスクリプトをプレーヤーの横に表示して、音声テキストをビデオに表示できます。 ユーザーが話している単語を聞くと、トランスクリプト内の一致する単語に下線が引かれるか、強調表示されます。 インタラクティブなトランスクリプトを使用して、視聴体験を制御できます。 たとえば、視聴者はビデオのトランスクリプトを検索し、任意の単語をクリックして正確なポイントに移動できます。

ギャラリーインページエクスペリエンスを使用してビデオを公開している場合は、ビデオトランスクリプトを表示するために3Playコンポーネントをエクスペリエンスに追加できます。 詳しくは、 インページエクスペリエンスへのコンポーネントの追加.

注：このトピックでは 3Playプラグイン そしてあなたがあなたのキャプション/トランスクリプトを生成するために3Playメディアを使っていると仮定します。 Brightcove キャプション/トランスクリプトを生成できる他のパートナーもいます。 パートナーのリストについては、 このページ.

注意：3Play Mediaインタラクティブトランスクリプトとの連携に関する詳細については、 Brightcove Player、を参照してください。 3Playメディアサイト.

iFrame埋め込みを使用する

iFrame embedメソッドは、以下の関連するトランスクリプトでビデオプレーヤーを埋め込むために必要なiFrame埋め込みコードを提供します。 これは、コーディングを必要とせずに、プレイヤーにトランスクリプトを追加する簡単な方法です。

注：これらの手順は、あなたがすでにあなたの Video Cloud および3Play Mediaアカウント。 詳細については、 3Playメディアサイト.

プレーヤーとトランスクリプト用に埋め込まれたiFrameを生成するには、次の手順に従います。

  1. あなたの3Playアカウントにログインします。
  2. トランスクリプトを表示するビデオを見つけて、タイトルをクリックします。
  3. ナビゲーションヘッダーの 発行>発行プラグイン.
  4. 拡大する ビデオ設定 の項目を検索します。
  5. 入力する プレーヤーID Brightcove Player にビデオを公開したいです。 の 動画ID や 口座番号 選択したビデオIDを入力します。 Video Cloud 口座番号。

    注：この例では、ビデオは Brightcove デフォルトプレーヤー.

  6. の中に プラグイン機能 セクション、選択 インタラクティブトランスクリプト 次に適切なオプションを選択します。
  7. 拡大する スタイル設定 セクションとセット 幅 や 身長 プラグイン用。 幅と高さには、埋め込まれたビデオとトランスクリプトの両方が含まれることに注意してください。
  8. 埋め込みコードをコピーして、HTMLページに貼り付けます。

するオプションがあります 新しいテンプレートを保存。 3Play Mediaはプラグインテンプレートを保存する機能を提供します。 その後、次のビデオでは、毎回プラグインを設定する代わりにテンプレートを選択できます。 また、テンプレートに加えられた変更は、公開されているすべてのプラグイン内に反映されるため、複数のプラグインの外観を更新する方が簡単です。 プラグインテンプレートを使った作業の詳細については、3Playのドキュメントを参照してください。 プラグインテンプレートの作成と管理.

JavaScript埋め込みを使用する

JavaScript埋め込みオプションは、iFrame埋め込みメソッドよりも多くのカスタマイズを提供します。 この方法では、JavaScriptとCSSの基本的な理解が必要です。 JavaScript埋め込みコードには含まれません Brightcove Player だからプレイヤーはからのコードを埋め込む Video Cloud 別の手順としてページに追加する必要があります。 動画を公開するときは、アドバンストプレーヤの埋め込みコードを使用する必要があります。

注：これらの手順は、あなたがすでにあなたの Video Cloud および3Play Mediaアカウント。 詳細については、 3Playメディアサイト.

トランスクリプト用のJavaScript埋め込みを生成するには、次の手順に従います。

  1. あなたの3Playアカウントにログインします。
  2. トランスクリプトを表示するビデオを見つけて、タイトルをクリックします。
  3. ナビゲーションヘッダーの 発行>発行プラグイン.
  4. 拡大する ビデオ設定 の項目を検索します。
  5. 　 埋め込み方法選択 カスタム（Javascript）.
  6. 確認する ビデオプレーヤーの種類 is ブライトコーブ ビデオターゲット プレーヤーの埋め込みコードに値を追加する必要があります（後の手順で行います）。
  7. の中に プラグイン機能 セクション、選択 インタラクティブトランスクリプト 次に適切なオプションを選択します。
  8. 埋め込みコードをコピーして、HTMLページに貼り付けます。
  9. の中に スタイル設定 セクション、コピー id クリップボードへのテキスト。 この値はプレーヤーの埋め込みコードに追加されます。
  10. （Mediaモジュールから）HTMLページのプレーヤ埋め込みコードを編集して、 id へのパラメータ <video> タグ。 埋め込みコードの例を以下に示します。

    注意：標準のiframe埋め込みコードではなく、Advanced埋め込みコードを使用する必要があります。

その後、CSSを使用してトランスクリプトをスタイルすることができます。 たとえば、下のCSSはトランスクリプトボックスの幅を設定し、現在の単語の色を変更し、トランスクリプトで使用されているテキストとフォントの色を変更します。

 

<style type = "text / css">

.p3sdk-interactive-transcript {
幅：640px。
}
.p3sdk-current-word {
background：red！important;
}
.p3sdk-interactive-transcript-content p {
色：オレンジ！重要;
フォントスタイル：斜体！important;
font-size：20px！important;
}
</スタイル>
