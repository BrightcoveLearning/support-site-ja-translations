ビデオパフォーマンスの表示

ビデオパフォーマンスセクションは、内部で再生されたビデオの高レベルのビデオパフォーマンスを提供します Audience対応プレイヤー。

注： Audience 分析システムは一般的な Video Cloud 分析システム、および2つのデータの追跡方法が異なります。 したがって、2つを比較すると、一部のメトリックに違いが見られます。 たとえば、 Video Cloud 分析システムは、ビデオが開始するとすぐにビューをカウントしますが、 Audience 分析システムは、ビデオの10秒まで再生回数をカウントしません。 したがって、14視聴者が動画を開始しても、そのうち6秒が再生される前に10視聴者が停止すると、 Video Cloud 分析では14ビューが表示されますが、 Audience モジュールは8を表示します。

を選択して開始 期間 チャート用。 次のオプションが利用可能です。

  • 7日
  • 30日
  • 90日

次のグラフが表示されます。

  • 最も視聴されたビデオ
  • 最も魅力的な動画
  • 視聴者の傾向

最も視聴されたビデオ

最も再生されたビデオのグラフには、最も再生されたビデオが表示されます Audience選択した期間に対応したプレーヤー。

最も魅力的な動画

[最も魅力的な動画]チャートには、最も魅力的な動画が表示されます。 Audience選択した期間の有効になっているプレーヤー。 エンゲージメントは、平均視聴率です。 ビデオエンゲージメントの計算方法の詳細については、トレーニングビデオをご覧ください。 動画エンゲージメントの計算.

視聴者の傾向

ビューアトレンドチャートには、選択した期間のビデオビューが経時的に表示されます。 既知のビューアーと匿名のビューアーの両方からのビューが表示されます。 グラフにカーソルを合わせると、実際のデータポイントが表示されます。

注：Marketo Munchkin接続を使用している場合、ビューアートレンドグラフには不明なビューアーのデータのみが表示されます。 マンチキン接続を使用している場合、ユーザーの詳細は収集されません。 ユーザー情報の詳細については、Marketo REST接続を使用する必要があります。 詳細については、 統合 Video Cloud REST APIを使用したMarketoの使用.

Audience 漏斗

挽き目 Audience ファネルは、ビデオのメトリックを表示します Audienceに対応したプレーヤー Audience 選択した期間のリードフォーム：

  • ビデオのインプレッション - 読み込まれた動画の数
  • 動画の再生回数 -ビデオが再生を開始した回数。ストリームの開始時に記録されます（巻き戻しや再生は含まれません）。 個々の視聴者の測定ではありません
  • リードフォームビュー -リードフォームビューの総数
  • 変換 -提出されたリードフォームの総数

トップ変換ビデオ

コンバージョン率の高い動画のグラフには、コンバージョン率が最も高い動画が表示されます。 コンバージョン率は、送信されたリードフォームの数を動画のインプレッション数で割って計算されます。

リードチャート

Leads Chartには、選択した期間の提出済みリードフォームの数が日別に表示されます。 グラフにカーソルを合わせると、実際のデータポイントが表示されます。

データ接続のセットアップ

挽き目 データ接続 オプションは、 Audience 統合。 Audience は、一般的なマーケティング自動化プラットフォーム（「MAP統合」）と統合できます。または、表示データをローカルに保存してから、エクスポートまたは抽出して Audience API （「非MAP」統合）。 次の表に、サポートされている統合の概要を示します。

MAP統合 非MAP統合
  • Oracle Eloqua
  • Marketo（REST API）
  • Marketo（マンチキン）
  • Salesforce
  • HubSpot（REST API）
  • HubSpot（クライアントサイドAPI）
  • カスタム統合
  • シングル・サインオン
  • Audience 追跡

公開するとき Audience有効なプレーヤーは、高度な埋め込み（ページ内）パブリッシングコードを使用します。 標準埋め込み（iframe）公開コードは、 Audience プラグインはiframe内で実行されるため、親ページのURLパラメータにアクセスできず、正しいCookieを取得できない可能性があります。

注：構成できる接続の数は、 Video Cloud サブスクリプション。

同期アクティビティの表示

挽き目 同期アクティビティ オプションは、同期された最新の10個のジョブを表示します Audience マーケティング自動化プラットフォームに。 同期はXNUMX時間ごとに実行されます。 接続を選択してから、 テスト同期 同期を実行するボタン。 これは、イベントデータがマーケティングオートメーションプラットフォームに送信されているかどうかを確認するのを待つ必要がないように、新しい統合をテストするときに役立ちます。 Test Syncは5分に10回、XNUMX時間にXNUMX回まで実行できます。

注：カスタム統合を使用する場合、同期ジョブは作成されません。 同期化ジョブは、マーケティングオートメーションプラットフォームとの統合を使用する場合にのみ作成されます。
