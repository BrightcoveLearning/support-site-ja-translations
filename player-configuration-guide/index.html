このトピックでは、プレーヤーの設定を表示および変更する方法について説明します。 JSONの構文も文書化されています。

概要

プレーヤーの設定で設定できるプロパティは、以下で詳しく説明します（左のナビゲーションにリストされています）。 アカウントID。 また、この文書には、プロパティー値の表示および構成方法に関する情報も含まれています。

あなたは使用しないでください data-setup 　 Brightcove Player。 あなたは data-setup APIドキュメントでは、これは、ドキュメンテーションがVideo.jsプレーヤーのソースコードから直接生成され、その属性を video タグを付けてください。 この属性は設定情報をプレーヤーに送信しますが、 Brightcove Player 別の方法を使用してこのタスクを実行します。 data-setup 信頼できない。

設定を表示

このセクションでは、プレーヤーの設定を表示するさまざまな方法があります。

Studioの使用

Studioを使用して、プレーヤーのJSON設定を表示できます。 次の手順は、プロセスの詳細です。

  1. Video Cloud Studioで プレイヤー モジュールを開き、構成を表示するプレーヤーを探します。
  2. プレーヤーのリンクをクリックしてプレーヤーのプロパティを開きます。
  3. ナビゲーションヘッダーの JSONエディタ 左側のナビゲーションメニュー。
  4. 次のようなプレーヤーの設定を表示できます。
変更を行って設定を保存する場合は、非常に注意してください。

を使用したブラウジング config.json

新しい Brightcove プレーヤーの設定はJSONファイルに保存されます。 このJSONファイル、またはその一部をいくつかの方法で見ることができます。

プレーヤーの設定を表示する1つの方法は、プレーヤーのURLを index.html 〜へ config.json。 例えば：

http://players.brightcove.net/1507807800001/HJyMlHiuZ_default/config.json

プレーヤー設定のプロパティの中には、次のものがあります。

  • プレーヤーの名前とバージョン
  • プレーヤーが再生するビデオ
  • プレーヤーが所属するアカウント
  • プレーヤーのID
  • プレイヤーが使用できるすべてのプラグイン、およびプラグインコードの場所のURL
  • ビデオ再生の前に表示するポスター

名前/説明の設定

プレーヤーの設定を変更するには、 /$ACCOUNT_ID/players/$PLAYER_ID/configuration それはこの文書の重点である。 ただし、2つのプロパティを変更するだけで /$ACCOUNT_ID/players/$PLAYER_ID 終点。 これらの2つの特性は name や description.

注： name あなたが入力したプレイヤーは player_name あなたが置くことによって構成を得るとき config.json プレーヤーのプレビューURLの最後に表示されます（ name あなたが Player Management API。 しかし、あなたは name プレーヤーを作成または更新するときに使用します。

curl \
--header "Content-Type: application/json" \
--user $EMAIL \
--request PATCH \
--data '{
  "name": "My New Player Name",
  "description": "My new player description"
}' \
